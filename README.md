### ğŸµ Play MIDI generated by LilyPond

#### Using Qsynth

- Install Qsyth and any sound found you like
- Install your distro's package providing the `pmidi` command
- Launch Qsynch and configure the sound font
- Run `pmidi kept.midi`

With this approach, you can simultaneously connect a MIDI keyboard and have
QSynth play both the notes from your keyboard and the LilyPond-generated midi
file.

#### Using Timidity

- Install and configure [`timidity++`](https://wiki.archlinux.org/index.php/Timidity%2B%2B)
- Compile the notes
- Run `timidity kept.midi`
